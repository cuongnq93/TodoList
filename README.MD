### TodoList

#### Requirements

```
PHP 7.1 or newer
Mysql 5.7.26
Nodejs 12 or newer
```

#### Install

* Config
Edit `app/config.php`

* Run command
```html
composer install
composer dumpautoload
yarn install
yarn build
```

* Import Database
```
Import database.sql to your database
```

* Test
```
./vendor/bin/phpunit
```
